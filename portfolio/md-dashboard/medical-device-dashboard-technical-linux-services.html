An example of python code used to develop the linux services.

I was responsible for implementing many linux backend services written in python that subscribed to the topics and manipulated the SQLITE databases to update, insert, delete, or upsert data. One such service was created to monitor and configure the underlying linux network manager through DBUS. A major service also uses queues, and multithreading to handle new incoming requests while composing and sending emails that contain HTML template bodies translated via gettext to supported languages with attachments in text, pdf or zip form.
